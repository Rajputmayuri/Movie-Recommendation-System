# Movie-Recommendation-System
A machine-learning based recommendation system that suggests movies to users based on similarity, user ratings, and content features. This project uses the MovieLens dataset and applies data preprocessing, feature engineering, and recommendation algorithms to generate personalized movie suggestions.
